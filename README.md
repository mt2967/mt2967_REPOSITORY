This paper covers PC software development techniques in the field of factory automation (FA).
The development targets are standalone image inspection machines and measurement devices operating continuously within factories.
These systems are difficult to achieve using off-the-shelf general-purpose inspection machines or general-purpose programs alone, and are envisioned to include several development elements.
Keywords: Industrial PC, C++/CLI, Multithreading, Image Inspection, Prime Factorization, ADS-B
